krazedkrish's awesome config
===========

My configs are based on

* [mrzapp's awesomerc](https://github.com/mrzapp/awesomerc)
* [setkeh's Awesome-3.5](https://github.com/setkeh/Awesome-3.5)

## Features
- Gnome fonts
- Gnome icons
- Wifi widget
- Battery widget
- Volume widget
- Brightness widget
